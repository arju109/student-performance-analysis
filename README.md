📊 Student Performance Analysis

 Objective

Analyze student data to identify key factors affecting academic performance.

 Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

 Analysis

- Study time vs marks
- Absences vs performance
- Failures impact
- Correlation between features

 Key Insights

- Higher study time improves performance
- Absences negatively affect marks
- Previous grades strongly influence final score

 How to Run

1. Open notebook in Google Colab
2. Upload dataset
3. Run all cells

 Skills Demonstrated

- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Feature Engineering
